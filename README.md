# 🎮 Number Guessing Game (C++)

A simple console-based number guessing game built using C++ as part of my learning journey in programming and problem-solving.

## 🔹 Features
- Random number generation
- User input handling
- Hints for higher or lower guesses
- Attempt counter

## 🛠️ Concepts Used
- Loops
- Conditional statements
- Functions
- Standard input/output
- Random number generation

## 🚀 How to Run
1. Compile the program using a C++ compiler  
2. Run the executable in the terminal  
3. Follow on-screen instructions

## 📌 Purpose
This project was built to strengthen my understanding of C++ fundamentals and logical thinking.
